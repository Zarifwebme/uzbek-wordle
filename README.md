# O'zbekcha Wordle

O'zbekcha Wordle - bu mashhur Wordle o'yinining o'zbek tilidagi versiyasi.

## Qanday o'ynaydi?

- 🎯 5 harfli o'zbek so'zni toping
- 🔢 Sizda 6 ta urinish bor
- 🎨 Harf ranglari:
  - 🟩 **Yashil** - to'g'ri harfni to'g'ri joyda topganid
  - 🟨 **Sariq** - harfni soyda bor, lekin noto'g'ri joyda
  - ⬜ **Oq** - bu harfni soyda yo'q

## Texnologiya

- HTML5
- CSS3
- Vanilla JavaScript
- Bootstrap 5

## Fayllar

- `index.html` - Asosiy sahifa
- `app.js` - O'yin mantiqasi
- `style.css` - Uslublar

## Ishlatish

Faylni brauzerda oching yoki lokal serverda ishga tushiring va ko'rsatmaga amal qiling:

```bash
python -m http.server
```

Keyin `http://localhost:8000` ga boring.
